# دليل تعديل نصوص التيليجرام

## 📝 طريقة التعديل

يمكنك تعديل نصوص الإشعارات بسهولة من ملف `telegram_texts.json`

### 🚀 خطوات التعديل:

1. **افتح ملف** `telegram_texts.json`
2. **ابحث عن** نوع الإشعار الذي تريد تعديله
3. **عدل النص** كما تريد
4. **احفظ الملف**
5. **أعد تشغيل النظام** أو اضغط زر "🔄 إعادة تحميل القنوات"

---

## 📋 المتغيرات المتاحة

استخدم هذه المتغيرات في النصوص (بداخل `{}`)

### متغيرات عامة:
- `{symbol}` - رمز الشركة (SPX, SPY, QQQ, NDX)
- `{type}` - نوع العقد (CALL أو PUT)
- `{contract}` - اسم العقد الكامل
- `{strike}` - سعر الإضراب (Strike)
- `{time}` - الوقت

### متغيرات الأسعار:
- `{entry_price:.2f}` - سعر الدخول (بخانتين عشريتين)
- `{exit_price:.2f}` - سعر الخروج
- `{current_price:.2f}` - السعر الحالي
- `{highest_price:.2f}` - أعلى سعر
- `{bid:.2f}` - سعر العرض (Bid)
- `{ask:.2f}` - سعر الطلب (Ask)
- `{last:.2f}` - آخر سعر

### متغيرات الربح/الخسارة:
- `{profit:.2f}` - مبلغ الربح
- `{profit_pct:.1f}` - نسبة الربح المئوية
- `{pnl:.2f}` - الربح/الخسارة (P&L)
- `{profit_emoji}` - إيموجي تلقائي (✅ للربح / ❌ للخسارة)
- `{profit_status}` - نص جاهز (ربح: +$XX أو خسارة: -$XX)

### متغيرات خاصة:
- `{drop_from_peak:.1f}` - النزول من القمة (trailing stop)
- `{protection_level:.1f}` - مستوى حماية رأس المال
- `{reason}` - سبب الإغلاق أو الرفض

## 🎨 تنسيقات HTML المدعومة

يمكنك استخدام تنسيقات HTML في النصوص:

- `<b>نص عريض</b>` - **نص عريض**
- `<i>نص مائل</i>` - *نص مائل*
- `<code>كود</code>` - `كود`
- `<u>نص مسطر</u>` - نص مسطر
- `<s>نص مشطوب</s>` - ~~نص مشطوب~~

## 📌 أمثلة على التعديل

### مثال 1: تعديل رسالة فتح الصفقة

**قبل:**
```json
"position_opened": "✅ <b>دخول {type}</b>\n..."
```

**بعد:**
```json
"position_opened": "🚀 <b>تم فتح صفقة {type}</b>\n📍 العقد: {contract}\n💰 السعر: ${entry_price:.2f}\n⏰ {time}"
```

### مثال 2: تعديل رسالة ضرب الهدف

**قبل:**
```json
"profit_target_hit": "🎯🎯 <b>ضرب الهدف!</b>\n..."
```

**بعد:**
```json
"profit_target_hit": "🎉🎯 <b>تحقيق الهدف بنجاح!</b>\n💵 ربح: <b>${pnl:.2f}</b>\n📊 النسبة: <b>{profit_pct:.1f}%</b>\n⏰ {time}"
```

---

## ⚙️ التنسيق الصحيح للأرقام

- `{price:.2f}` - رقم بخانتين عشريتين (مثال: 51.50)
- `{price:.1f}` - رقم بخانة عشرية واحدة (مثال: 51.5)
- `{price:.0f}` - رقم بدون خانات عشرية (مثال: 52)
- `{price:+.2f}` - رقم مع علامة + أو - (مثال: +12.50 أو -5.25)

---

## ⚠️ ملاحظات مهمة

1. **احفظ الملف بترميز UTF-8** لدعم العربية
2. **لا تحذف المتغيرات** `{}` - غيّر النص حولها فقط
3. **استخدم `\n`** للسطر الجديد
4. **الفواصل بين الأسطر** `━━━━━━━` اختيارية (يمكن حذفها أو تغييرها)
5. **الإيموجي** يعمل بشكل طبيعي ✅
