# SPX Smart - نظام تداول الخيارات الاحترافي

نظام متقدم لتداول خيارات SPX مع واجهة رسومية كاملة، مراقبة لحظية، وإدارة مخاطر ذكية.

## 🌟 المميزات الرئيسية

### 📊 قائمة المراقبة الحية
- **25 عقد CALL + 25 عقد PUT** لكل شركة
- تحديث تلقائي كل **10 ثواني**
- عرض Strike, Bid, Ask لكل عقد
- دعم عدة شركات (SPX, NDX, SPY, QQQ)

### 🎯 نطاق الاختيار الذكي
- اختيار من نطاق **1-4** (قابل للتعديل)
- يختار **الأعلى سعراً** من النطاق تلقائياً
- فلترة حسب نطاق السعر ($3-$4 افتراضياً)

### 📈 التتبع اللحظي
- تحديث كل **0.5 ثانية**
- تتبع أعلى سعر وصل له العقد
- حساب P&L فوري
- إشعارات عند كل قمة جديدة

### ⚠️ إدارة المخاطر المتقدمة
1. **🛑 وقف الخسارة (Stop Loss)**
   - نسبة مئوية أو مبلغ ثابت
   - إغلاق تلقائي عند الوصول

2. **📊 وقف متحرك (Trailing Stop)**
   - يتحرك مع السعر الأعلى
   - حماية الأرباح المحققة

3. **🛡️ حماية رأس المال (Capital Protection)**
   - حد أقصى للخسارة اليومية
   - إيقاف التداول عند الوصول

4. **🎯 هدف الربح (Profit Target)**
   - إغلاق تلقائي عند تحقيق الهدف
   - تأمين الأرباح

### 📱 تيليجرام متقدم
- إشعارات فورية لجميع الأحداث
- صور احترافية مخصصة
- نصوص قابلة للتعديل
- قنوات متعددة لكل شركة

### 🌐 Webhook Server
- استقبال إشارات من TradingView
- **دعم تحديد عدد العقود في الإشارة**
- معالجة تلقائية فورية
- سجل كامل للإشارات
- راجع [WEBHOOK_GUIDE.md](WEBHOOK_GUIDE.md) للتفاصيل

### 🔌 حالة الاتصال
- مؤشر حي لحالة اتصال IBKR
- 🟢 متصل / 🔴 غير متصل
- تحديث تلقائي كل 3 ثواني

## 🚀 التثبيت والتشغيل

### المتطلبات
- Python 3.10 أو أحدث
- TWS أو IB Gateway
- حساب Telegram Bot (اختياري)

### خطوات التثبيت

#### 1️⃣ تثبيت المكتبات
```bash
# الطريقة السهلة
INSTALL.bat

# أو يدوياً
pip install -r requirements.txt
```

#### 2️⃣ إعداد ملف .env
```env
# Telegram
TELEGRAM_BOT_TOKEN=your_token_here
TELEGRAM_CHAT_ID=your_chat_id

# IBKR
IBKR_HOST=127.0.0.1
IBKR_PORT=7497          # 7497 للتجريبي، 7496 للحقيقي
IBKR_CLIENT_ID=10

# Flask
FLASK_HOST=0.0.0.0
FLASK_PORT=5000
```

#### 3️⃣ تشغيل النظام
```bash
# الطريقة السريعة
RUN.bat

# أو يدوياً
python START.py
```


## 📖 دليل الاستخدام

### تشغيل النظام

1. **افتح TWS/Gateway**
   - تأكد من تفعيل API Connection
   - Port: 7497 (تجريبي) أو 7496 (حقيقي)

2. **شغّل النظام**
   ```bash
   python START.py
   ```

3. **اضغط "▶️ تشغيل النظام"**
   - سيتصل تلقائياً بـ IBKR
   - يبدأ تحديث قائمة المراقبة
   - يفتح Webhook Server

### قائمة المراقبة

- **SPX افتراضياً** يتم تحميله تلقائياً
- **إضافة شركة جديدة**: اضغط "➕ إضافة شركة"
- **التحديث**: كل 10 ثواني تلقائياً
- **العرض**: Strike, Bid, Ask

### استقبال إشارة من TradingView

1. **إعداد Alert في TradingView**:
   ```
   Webhook URL: http://your-server:5000/webhook
   Message:
   {
       "symbol": "SPX",
       "type": "CALL",
       "action": "BUY"
   }
   ```

2. **النظام يقوم بـ**:
   - استقبال الإشارة
   - البحث عن عقد في النطاق (1-4)
   - إرسال إشعار تيليجرام
   - فتح الصفقة
   - بدء التتبع اللحظي

### إدارة المخاطر

#### مثال: وقف الخسارة 10%
1. اختر "🛑 وقف الخسارة"
2. الشركة: SPX
3. النوع: نسبة %
4. القيمة: 10
5. اضغط "✓ تطبيق"

#### مثال: هدف ربح $200
1. اختر "🎯 الهدف"
2. الشركة: SPX
3. النوع: مبلغ $
4. القيمة: 200
5. اضغط "✓ تطبيق"

### إغلاق صفقة يدوياً
1. انقر على الصفقة في "الصفقات النشطة"
2. اضغط "إغلاق الصفقة المحددة"
3. تأكيد الإغلاق

- ✅ تحقق من Subscription للبيانات
