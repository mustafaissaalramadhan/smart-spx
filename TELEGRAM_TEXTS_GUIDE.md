# دليل نصوص إشعارات التيليجرام

## 📝 نظرة عامة
النصوص الافتراضية للإشعارات مطابقة تماماً للنظام القديم (SPX). يمكنك تعديلها من خلال الواجهة الرسومية.

## 🎯 كيفية التعديل
1. افتح النظام
2. اذهب إلى قسم "إعدادات التيليجرام"
3. اضغط على زر "✏️ تعديل نصوص الرسائل"
4. عدّل النصوص كما تريد
5. احفظ التغييرات

## 💡 المتغيرات المتاحة

### للجميع:
- `{type}` - نوع الصفقة (CALL أو PUT)
- `{symbol}` - رمز الشركة (SPX, NDX, SPY, QQQ)
- `{time}` - الوقت

### فتح الصفقة (position_opened):
- `{contract}` - اسم العقد الكامل
- `{strike}` - سعر التنفيذ
- `{entry_price}` - سعر الدخول
- `{bid}` - سعر العرض
- `{ask}` - سعر الطلب

### تحديث السعر (price_update / new_high):
- `{contract}` - اسم العقد
- `{last}` - آخر سعر
- `{current_price}` - السعر الحالي (Bid)
- `{ask}` - سعر الطلب
- `{entry_price}` - سعر الدخول
- `{highest_price}` - أعلى سعر تم الوصول إليه
- `{profit}` - الربح بالدولار
- `{profit_pct}` - نسبة الربح المئوية
- `{profit_emoji}` - رمز تعبيري (💰 أو 📉)

### إغلاق الصفقة (position_closed):
- `{contract}` - اسم العقد
- `{entry_price}` - سعر الدخول
- `{exit_price}` - سعر الخروج
- `{highest_price}` - أعلى سعر
- `{profit}` - الربح/الخسارة
- `{profit_pct}` - النسبة المئوية
- `{reason}` - سبب الإغلاق (إشارة خروج / وقف خسارة)
- `{profit_status}` - نص حالة الربح (ربح أو خسارة مع التفاصيل)

### وقف الخسارة (stop_loss_hit):
- `{current_price}` - السعر الحالي
- `{stop_loss}` - سعر وقف الخسارة

### الملخص اليومي (daily_summary):
- `{total_profit}` - إجمالي الأرباح
- `{total_loss}` - إجمالي الخسائر
- `{net_profit}` - صافي الربح
- `{winning_trades}` - عدد الصفقات المربحة
- `{losing_trades}` - عدد الصفقات الخاسرة

## 📋 مثال على التنسيق

```
✅ <b>دخول {type}</b>
━━━━━━━━━━━━━━━━━━━━━━━
العقد: <code>{contract}</code>
Strike: <b>{strike}</b>
سعر الدخول: <b>{entry_price:.2f} USD</b>
⏰ الوقت: {time}
━━━━━━━━━━━━━━━━━━━━━━━
```

## 🎨 علامات HTML المدعومة
- `<b>النص</b>` - نص عريض (Bold)
- `<i>النص</i>` - نص مائل (Italic)
- `<code>النص</code>` - نص بخط برمجي (Code)
